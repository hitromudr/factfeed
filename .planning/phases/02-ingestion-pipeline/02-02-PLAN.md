<task type="auto">
  <name>Task 1: Create RSS feed fetcher with robots.txt compliance</name>
  <files>factfeed/ingestion/fetcher.py</files>
  <action>
Create `factfeed/ingestion/fetcher.py` with three async functions:

**`fetch_rss_feed(source: dict, client: httpx.AsyncClient) -> feedparser.FeedParserDict`:**
- Fetch RSS XML bytes using `client.get(source["feed_url"])` with `raise_for_status()`
- Parse with `feedparser.parse(response.content)` via `asyncio.get_event_loop().run_in_executor(None, feedparser.parse, response.content)` to avoid blocking the event loop
- If `feed.bozo` is True, log a WARNING with `structlog` including source name and bozo_exception — do NOT skip the feed (bozo means degraded, not broken)
- Return the feedparser dict regardless of bozo status — caller checks `len(feed.entries)`

**`fetch_article_page(url: str, client: httpx.AsyncClient) -> bytes | None`:**
- Fetch HTML with `client.get(url)` and `raise_for_status()`
- Return `response.content` (bytes)
- On any exception (httpx.HTTPError, httpx.TimeoutException, etc.): log WARNING with url and error string, return None
- Do NOT retry — the runner handles retries at the source level

**`can_fetch(url: str, user_agent: str, client: httpx.AsyncClient) -> bool`:**
- Parse domain from URL using `urllib.parse.urlparse`
- Maintain a module-level `_robots_cache: dict[str, RobotFileParser | None]` keyed by domain (scheme://netloc)
- On first call for a domain: fetch robots.txt with httpx (not urllib — to use our User-Agent), parse with `RobotFileParser.parse(lines)`. On fetch failure, cache None and return True (allow)
- On cached domain: call `rp.can_fetch(user_agent, url)` and return result
- Use `asyncio.get_event_loop().run_in_executor` for the synchronous `rp.can_fetch()` call

Use `structlog.get_logger()` for all logging. Import at module level.

  <name>Task 2: Create article extractor with trafilatura and partial fallback</name>
  <files>factfeed/ingestion/extractor.py</files>
  <action>
Create `factfeed/ingestion/extractor.py` with one main function:

**`extract_article(html_bytes: bytes, url: str, rss_summary: str | None) -> dict`:**
- Call `trafilatura.bare_extraction(html_bytes, url=url, include_images=True, favor_recall=True)` to extract body text and metadata
- If `result` is not None AND `len(result.get("text") or "") >= MINIMUM_BODY_LENGTH` (200 chars):
  - Return a dict with:
    - `body`: `result["text"]` (plain text for NLP in Phase 3)
    - `body_html`: result from `trafilatura.extract(html_bytes, url=url, output_format="html", include_images=False, favor_recall=True)` — minimal HTML with paragraph/blockquote tags preserved for Phase 4 display. If extract returns None, wrap `result["text"]` in `<p>` tags.
    - `author`: `result.get("author")` or None
    - `published_at`: `result.get("date")` or None (ISO format string — caller converts to datetime)
    - `lead_image_url`: `result.get("image")` or None
    - `is_partial`: False
- If result is None OR body text is too short:
  - Return partial fallback dict:
    - `body`: `rss_summary or ""`
    - `body_html`: `f"<p>{rss_summary or ''}</p>"`
    - `author`: None
    - `published_at`: None
    - `lead_image_url`: None
    - `is_partial`: True
- Wrap the entire extraction in try/except Exception — any unhandled error triggers partial fallback

Define `MINIMUM_BODY_LENGTH = 200` as a module constant.

Also create a helper `parse_article_date(date_str: str | None) -> datetime | None`:
- If date_str is None, return None
- Try `dateutil.parser.parse(date_str)` to handle various ISO/human date formats
- If parsing fails, return None
- Ensure result is timezone-aware (default to UTC if naive)

Use `structlog.get_logger()` for logging extraction failures at DEBUG level.

  <name>Task 3: Create URL deduplicator and source definitions</name>
  <files>factfeed/ingestion/deduplicator.py, factfeed/ingestion/sources.py</files>
  <action>
**Create `factfeed/ingestion/deduplicator.py`** with two functions:

**`compute_url_hash(url: str) -> str`:**
- Strip whitespace from URL
- Parse with `urllib.parse.urlparse`
- Normalize: lowercase scheme and netloc, keep path, strip params/query/fragment (removes tracking params)
- Reconstruct with `urlunparse`
- Return `hashlib.sha256(normalized.encode("utf-8")).hexdigest()`
- The result is a 64-char hex string matching the `url_hash` column's String(64)

**`async def article_exists(url_hash: str, session: AsyncSession) -> bool`:**
- Execute `select(Article.id).where(Article.url_hash == url_hash).limit(1)` on the async session
- Return `result.scalar_one_or_none() is not None`
- Import Article from `factfeed.db.models` and select from `sqlalchemy`
