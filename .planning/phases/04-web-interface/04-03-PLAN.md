<task type="auto">
  <name>Task 2: Create article template with inline highlighting and collapsible opinions</name>
  <files>factfeed/templates/article.html</files>
  <action>
    Create `factfeed/templates/article.html` extending `base.html`:

    1. Article header:
       - `<h1>{{ article.title }}</h1>`
       - `<div class="article-meta"><span class="source">{{ article.source.name }}</span> <span class="date">{{ article.published_at.strftime('%B %d, %Y') if article.published_at else '' }}</span></div>`
       - Back link: `<a href="/">&larr; Back to search</a>`

    2. Legend (color key):
       ```html
       <div class="legend">
         <span class="legend-item"><span class="sentence fact">Fact</span></span>
         <span class="legend-item"><span class="sentence opinion">Opinion</span></span>
         <span class="legend-item"><span class="sentence mixed">Mixed/Unclear</span></span>
       </div>
       ```

    3. Article body with inline highlighting:
       - If `non_opinion_sentences` is empty AND `opinion_sentences` is empty (no classification yet):
         ```html
         <div class="article-body"><p>{{ article.body }}</p><p class="note">Classification pending.</p></div>
         ```
       - Otherwise, render non-opinion sentences in order:
         ```html
         <div class="article-body">
           {% for sentence in non_opinion_sentences %}
             {% set conf_label = "High" if sentence.confidence >= 0.7 else ("Medium" if sentence.confidence >= 0.4 else "Low") %}
             <span class="sentence {{ sentence.label }}" title="{{ conf_label }} confidence">{{ sentence.text }} </span>
           {% endfor %}
         </div>
         ```

    4. Collapsible opinion section (after the main body):
       ```html
       {% if opinion_sentences %}
       <details class="opinion-section">
         <summary class="opinion-toggle">Show opinion content ({{ opinion_count }} sentence{{ 's' if opinion_count != 1 else '' }})</summary>
         <div class="opinion-content">
           {% for sentence in opinion_sentences %}
             {% set conf_label = "High" if sentence.confidence >= 0.7 else ("Medium" if sentence.confidence >= 0.4 else "Low") %}
             <span class="sentence opinion" title="{{ conf_label }} confidence">{{ sentence.text }} </span>
           {% endfor %}
         </div>
       </details>
       {% endif %}
       ```

    5. Confidence mapping: `>= 0.7` = "High", `0.4 - 0.69` = "Medium", `< 0.4` = "Low". NEVER show raw decimal (Pitfall 5 from research). The `title` attribute provides native browser tooltips. CSS `::after` pseudo-element (already in style.css from Plan 01) provides styled tooltips on hover.

    6. The `<details>` element is natively collapsed (no `open` attribute). No JavaScript needed for collapse/expand (Pattern 7 from research). This is HTML-native and accessible.
  </action>
