**calibrator.py:**
Create `factfeed/nlp/calibrator.py` with:

1. `TemperatureScaler` class:
```python
import numpy as np

class TemperatureScaler:
    def __init__(self, temperature: float = 1.0):
        self.temperature = temperature

    def calibrate(self, raw_score: float) -> float:
        if self.temperature == 1.0:
            return raw_score
        logit = np.log(raw_score / (1 - raw_score + 1e-8))
        scaled_logit = logit / self.temperature
        return float(1 / (1 + np.exp(-scaled_logit)))

    def fit(self, raw_scores: list[float], true_labels: list[int]) -> float:
        from scipy.optimize import minimize_scalar
        from sklearn.metrics import log_loss

        def nll(T):
            cal = [self._calibrate_with_T(s, T) for s in raw_scores]
            probs = [[1 - p, p] for p in cal]
            return log_loss(true_labels, probs)

        result = minimize_scalar(nll, bounds=(0.1, 10.0), method='bounded')
        self.temperature = result.x
        return self.temperature

    def _calibrate_with_T(self, raw_score: float, T: float) -> float:
        logit = np.log(raw_score / (1 - raw_score + 1e-8))
        scaled = logit / T
        return float(1 / (1 + np.exp(-scaled)))
```

Default temperature=1.0 means no scaling (passthrough). The `fit()` method finds optimal temperature on a calibration set.

**pipeline.py:**
Create `factfeed/nlp/pipeline.py` with:

1. `SentenceResult` dataclass:
```python
@dataclass
class SentenceResult:
    text: str
    position: int
    label: Literal["fact", "opinion", "mixed", "unclear"]
    confidence: float  # 0.0-1.0, calibrated
```

2. `classify_article(body: str, zs_pipeline: Callable, calibrator: TemperatureScaler | None = None, source_name: str = "") -> list[SentenceResult]`:
   - Calls `segment_article(body)` to get sentence spans
   - For each sentence span (with enumerated position):
     - Run `pre_filter_sentence(sent_span, source_name)`
     - If pre-filter returns a result: use its label and confidence
     - Else: call `classify_sentence(sent_span.text, zs_pipeline)` to get raw label and confidence
     - If calibrator provided: apply `calibrator.calibrate(raw_confidence)` to the transformer result
     - Clamp all confidence values to [0.05, 0.95] range (prevent false certainty)
     - Append `SentenceResult(text=sent_span.text, position=position, label=label, confidence=confidence)`
   - Return the list

3. `classify_article_async(body: str, zs_pipeline: Callable, calibrator: TemperatureScaler | None = None, source_name: str = "") -> list[SentenceResult]`:
   - Wraps `classify_article` in `asyncio.to_thread()` to avoid blocking the event loop
   - The transformer inference is CPU-bound; `to_thread()` runs it in a thread pool

**persist.py:**
Create `factfeed/nlp/persist.py` with:

1. `persist_sentences(article_id: int, results: list[SentenceResult], session: AsyncSession) -> None`:
   - Delete existing sentences for this article: `await session.execute(delete(Sentence).where(Sentence.article_id == article_id))`
   - If results is empty, commit and return
   - Bulk insert: `await session.execute(insert(Sentence), [{"article_id": article_id, "position": r.position, "text": r.text, "label": r.label, "confidence": r.confidence} for r in results])`
   - `await session.commit()`
