<objective>
Implement the rule-based first layer of the hybrid classification pipeline: spaCy sentence segmentation and the pre-filter that routes attributed speech to "mixed" and short/ambiguous sentences to "unclear" without invoking the transformer.

Purpose: The pre-filter handles deterministic cases cheaply (~5-20ms/sentence vs ~200ms for transformer) and is a prerequisite for the full pipeline. Attribution detection (NLP-04) and ambiguous content flagging (NLP-03) are specified as rule-based behavior that must happen before the transformer runs.
Output: segmenter.py, pre_filter.py, and their unit tests.
</objective>

**segmenter.py:**
Create `factfeed/nlp/segmenter.py` with:

1. Module-level spaCy model load:
```python
import spacy
_nlp = spacy.load("en_core_web_sm")
```

2. `get_nlp() -> spacy.Language` — returns the shared spaCy Language object (for DependencyMatcher/PhraseMatcher initialization in pre_filter).

3. `segment_article(body: str) -> list[spacy.tokens.Span]` — parses the body text and returns `list(doc.sents)`. This is the single entry point for segmentation. Parse the body ONCE; do not re-parse per sentence.

**pre_filter.py:**
Create `factfeed/nlp/pre_filter.py` with:

1. Import the shared nlp object from segmenter: `from factfeed.nlp.segmenter import get_nlp`.

2. Define a `SentenceResult` dataclass:
```python
from dataclasses import dataclass
from typing import Literal, Optional

Label = Literal["fact", "opinion", "mixed", "unclear"]

@dataclass
class PreFilterResult:
    label: Label
    confidence: float
    reason: str  # "attribution", "short_sentence", "satire_marker", "breaking_news"
```

3. Initialize matchers at module level (run once when module loads):

**DependencyMatcher** for attribution verbs:
- Attribution verbs (lemmas): say, tell, claim, state, announce, report, allege, argue, warn, explain, note, add, confirm, deny, insist, suggest, indicate, assert, contend, maintain, remark, declare
- Pattern: anchor_verb (LEMMA in attribution_verbs, POS=VERB) with child nsubj via ">" relation
- Use `get_nlp().vocab` for matcher initialization

**PhraseMatcher** for attribution phrases:
- Phrases: "according to", "told reporters", "sources said", "sources say", "officials said", "officials say", "a spokesperson said", "in a statement"
- Use `attr="LOWER"` for case-insensitive matching

**Satire markers set:**
```python
SATIRE_MARKERS = {
    "the onion", "babylon bee", "reductress", "the daily mash",
    "the beaverton", "world news daily report", "satire", "[satire]",
}
```

**Breaking news patterns:**
```python
BREAKING_PATTERNS = ["breaking:", "breaking news:", "developing story:", "just in:"]
```

4. `is_attribution(sent_span: spacy.tokens.Span) -> bool`:
- Convert span to doc via `sent_span.as_doc()`
- Run both dep_matcher and phrase_matcher on the doc
- Return True if either matches

5. `is_unclear(sent_span: spacy.tokens.Span, source_name: str = "") -> bool`:
- Check `len(sent_span) < 30` (spaCy token count, NOT word count)
- Check if sentence text contains any satire marker (case-insensitive)
- Check if sentence text starts with any breaking news pattern (case-insensitive)
- Check if source_name is in SATIRE_MARKERS
- Return True if ANY check passes

6. `pre_filter_sentence(sent_span: spacy.tokens.Span, source_name: str = "") -> Optional[PreFilterResult]`:
- First check `is_attribution(sent_span)` → return `PreFilterResult("mixed", 0.6, "attribution")`
- Then check `is_unclear(sent_span, source_name)` → return `PreFilterResult("unclear", 0.1, reason)` where reason is the specific gate that triggered
- Otherwise return `None` (sentence needs transformer classification)

**Important:** Attribution check runs BEFORE unclear check. A 10-token sentence like "He said the deal is done." should be classified as "mixed" (attribution), not "unclear" (short).
